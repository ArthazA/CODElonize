# CODElonize Development Roadmap

The project should be implemented incrementally. Each phase should produce a working application before moving to the next phase.

=========================================================
PHASE 0 — PROJECT SETUP
=======================

□ Create Xcode project
□ Configure Git repository
□ Create folder structure
□ Create documentation
□ Import RealityKit and ARKit
□ Verify application launches

Goal:
A clean project foundation.

---

=========================================================
PHASE 1 — CORE AR
=================

□ Initialize ARSession
□ Plane detection
□ Host places island
□ Load island model
□ Display virtual pinpoints
□ Verify island remains stable

Goal:
A single-player AR scene.

---

=========================================================
PHASE 2 — MULTIPLAYER NETWORKING
================================

□ Create lobby
□ Join lobby
□ Generate lobby code
□ Peer discovery
□ Ready system
□ Start game synchronization

Goal:
Multiple devices connect successfully.
